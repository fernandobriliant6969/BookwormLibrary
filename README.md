## Fitur Aplikasi

- Login & Register
- Role Superadmin, Admin, Member & Guest
- Manajemen Buku dengan Fitur Search & Filter serta Upload Cover Buku menggunakan Sistem Cloudinary Image Cloud
- Manajemen Peminjaman dengan Fitur Search & Filter
- Manajemen Genre
- Manajemen User dengan Fitur Search & Filter
- Dashboard Admin & Member
- Sweetalert2 Animation
- Responsive Design
- Sistem Review
- Grafik Data menggunakan Chart.JS

## Teknologi yang Digunakan

- PHP
- Laravel
- Laravel Breeze
- MySQL
- GroqAI
- Bootstrap CSS
- Cloudinary
- Vite
- Laragon
